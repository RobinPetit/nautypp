Detailed Description

template<typename T>
class nautypp::ContainerBuffer< T >

Communication buffer between producer and consumer threads.

The buffer is separated into a read buffer and a write buffer. The consumer thread pops from the former, while the producer thread inserts in the latter. When the read buffer is empty, the consumer will swap the two buffers.
